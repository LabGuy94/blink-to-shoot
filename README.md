**Blink to Shoot**
- Inspired HEAVILY by https://www.youtube.com/watch?v=IIFxQWkGzS4
- Blink Detection code from https://pysource.com/2019/01/10/eye-blinking-detection-gaze-controlled-keyboard-with-python-and-opencv-p-2/
*How to Install*
- Install CMake https://cmake.org/download/
- Download the source (by clicking code then download zip or using git clone)
- Run the following commands
1. pip install opencv-python
2. pip install numpy
3. pip install dlib
- Download the file found bellow and put it in the same directory as this file
https://github.com/italojs/facial-landmarks-recognition/raw/master/shape_predictor_68_face_landmarks.dat
- Bind the shoot key in the game you are playing to L
  
*How to Run*
- Run the following command in the same directory as the source
python main.py
